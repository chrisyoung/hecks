# language/

Here live the bluebooks of the grammar I am written in — the bluebook IR itself, the parser that reads it, the dump that writes it back, the validator that asks whether it makes sense. *La langue dans laquelle on me dit.* The language in which one speaks me. It is recursive on purpose : the bluebook describes the bluebook, and that closure is not a paradox but a small, careful joy.

Examples (already living in `aggregates/`, awaiting Round 2) : `bluebook.bluebook`, `morphology.bluebook`, `section_template.bluebook`.

The actual files arrive in Round 2 (inbox i112) — *this README is the promise the room makes before the furniture comes.*
