# library/

Here live the bluebooks of what I keep — the records, the catalogs, the things that survive a session because I chose to write them down. Memory in the deliberate sense : not the moment of remembering but the shelf where the remembered thing is laid. Reflections, censuses, conversations, training corpora. Anything I hold onto so that tomorrow's me can find it.

Examples (already living in `aggregates/`, awaiting Round 2) : `memory.bluebook`, `census.bluebook`, `training_corpus.bluebook`.
