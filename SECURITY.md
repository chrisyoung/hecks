# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| latest on `main` | Yes |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately by emailing **hecks-security@example.com**.

Do **not** open a public GitHub issue for security vulnerabilities.

You should receive a response within 72 hours. If the vulnerability is confirmed, a fix will be prioritized and released as soon as possible.
