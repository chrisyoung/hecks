## Operating notes for next session

- **Heki files are always dirty** — `hecks_conception/information/*.heki` are live runtime state. Stash or ignore.
- **The Rust fixtures_parser needs each `fixture` directive on a single line** (i57). Multi-line silently drops attrs. Known workaround.
- **Antibody hook blocks non-DSL file commits** — exemption markers go in commit messages, not pre-written in files.
- **`bin/specialize <target> --diff`** is the quickest way to check byte-identity without running cargo.
- **Parity suite, fixtures suite, hecksagon suite** — each has its own `*_known_drift.txt` list. `specializer.hecksagon` is on hecksagon_known_drift (Ruby `:fs` treated as persistence).
