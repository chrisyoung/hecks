# Rails (Full)

Bundles ActiveHecks + HecksLive — one gem for Rails apps

## Install

```ruby
# Gemfile
gem "hecks_on_rails"
```

Add the gem and it auto-wires on boot. No configuration needed.

## Usage

```ruby
rails generate active_hecks:init
# That's it. Everything wires up.
```
